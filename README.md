# KitPass-ios.26-HealthKit-FoundationModelFramework
KitPass is a privacy-first iOS application. Built with SwiftUI and integrated with Apple HealthKit, KitPass provides a seamless experience for managing your health history for travelers. All using the "foundation model framework" in ios.26, the app access your health record and analyze using generative ai capbility from 3 billion on device LLM on your iphone.
